# izuna's BeatSaber patchnote discord bot

## Installation

```bash
npm install
```

## ENV

```env
TOKEN="BOT TOKEN"
WEBHOOKS="WEBHOOKS LINK"
```

## Start

```bash
npm run server
npm run start
node index.js
```

## Commands

```Chat
p!phs - Sending patchnote to Message channel

/ps - Sending patchnote to Interaction channel

/psw - Sending patchnote to Webhooks channel

```
